# 短连接生成平台

## 简介

本项目是基于express和mongodb的短连接生成平台。

# 使用方式

## 服务端安装

```bash
cd server
pnpm i
```

## 服务端配置 & 运行
进入config目录,修改mongodb的连接信息，然后运行以下命令

```bash
pnpm start
```
## 客户端安装

```bash
cd src
pnpm i
```

## 客户端运行

```bash
pnpm serve
```

## 访问

```bash
http://localhost:8080/
```

## 注意事项

本项目使用mongodb作为数据库，请先安装mongodb并启动。
